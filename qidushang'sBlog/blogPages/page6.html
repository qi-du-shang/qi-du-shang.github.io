Vue的v-once,v-pre,v-text,v-bind指令

<h2 v-once>{{message}}</h2>

当属性发生改变时,加v-once指令可以使原始展示属性不发生改变。

v-pre指令

使{{message}}以当前状态展示,不进行编译。

v-text指令

和正常编译类似,但标签中的内容会被v-text中属性的值覆盖。

<h2 v-text="message">,你好!</h2>

v-bind指令

动态绑定标签的属性。

<div id="app"><a v-bind:href="aHref">百度一下</a></div>

const app = new Vue({ el :'#app', data: { message: '你好啊', imgUrl: '1.png', aHref: 'http://www.baidu.com' } })
