Vue的v-cloak,v-html指令

运行时不会产生原始{{message}},未编译前的样子

[v-cloak]{ display: none; }

<div id="app" v-cloak>{{message}}</div>

v-html指令

<h2 v-html="url"></h2>

const app = new Vue({ el: '#app', data: { message: '你好啊', url: '< href="http://www.baidu.com">百度一下></a>' } })

当data里的属性有html标签时,v-html可以将这些标签编译,以编译后的状态展示出来。
