## 📖 项目简介
本项目是一套**基于 GitHub 生态实现的零服务器、纯静态自动化监控方案**，专为《洛克王国世界》远行商人（蛇瞳）数据查询打造。

采用「**云端定时脚本 + 静态数据文件 + 前端纯页面展示**」的极简架构，依托 GitHub Actions 实现无人值守数据爬取、微信告警推送，同时完全保留原有前端页面与访问链接，无需复杂运维，稳定低成本运行。
